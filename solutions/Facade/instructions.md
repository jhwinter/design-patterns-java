# Home Theater System Application Exercise

You are designing a home theater system that integrates multiple components such as a DVD player, projector, sound system, and lighting control. To simplify the interaction with these complex subsystems, you decide to implement a Facade pattern.



**Requirements**:

* Create a DVDPlayer class with methods to play, pause, and stop the DVD.
* Create a Projector class with methods to turn on, off and adjust settings.
* Create a SoundSystem class with methods to turn on, off and set volume.
* Create a LightingControl class with methods to turn on, off and dim lights.
* Implement a HomeTheaterFacade class that provides a simple interface to control the entire home theater system.



**Instructions**:

* Implement the individual components of the home theater system.
* Implement the HomeTheaterFacade that will interact with the subsystems.
* Demonstrate the use of the HomeTheaterFacade to create a movie-watching experience.



**Output**:

After executing the code, it will simulate the following sequence of operations:

* Display a message indicating that the user is getting ready to watch a movie.
* Dim the lights to the specified percentage.
* Turn on the projector.
* Set the projector's input to DVD.
* Turn on the sound system.
* Set the sound system volume to the specified level.
* Play the DVD.
* Display a message confirming that the movie is ready to watch.

Each operation should be clearly reflected in the output, confirming that the home theater system is set up correctly.
